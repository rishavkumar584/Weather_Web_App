# Weather_Web_App
A simple web app used to know the weather of a place.

# TechStack

## Core:
- React.js
- Vite
- Javascript
- Tailwind CSS
- Framer Motion
- Axios

# Folder Structure
- weather-dashboard
- /src  :
    - App.jsx
    - index.css
    - main.jsx
      
- index.html
- tailwind.config.js
- postcss.config.js
- vite.config.js
  
- /public/assets/  : Image files used in the project.
